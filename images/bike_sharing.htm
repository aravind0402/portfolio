As a road cyclist practitioner, I ride my bike a lot whenever possible rather than taking public transports. This is so enjoyable and healthy.

Nowadays, with the development of soft mobility in urban areas, do we see some patterns in the way people use it ?I’ll take the case of Seoul with its bike sharing system to do exploratory data analysis with Python.

The dataset can be found on UCI Machine Learning Repository: Seoul Bike Sharing Demand Data Set. It covers 2017 and 2018.

Let’s start exploring it ! First here are the Python libraries I will use for this EDA:
